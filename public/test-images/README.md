# SteelMotion 测试图片

这些图片只用于本地演示“图片到视频”的完整流程，均来自 Wikimedia Commons，并已裁切为 1920x1080 横版。

图片本身只放裸材料，不放加工线、成品车或动画效果；材料如何变成产品部件，由页面里的 prompt 引导视频模型生成。

| `raw-steel-slabs.jpg` | 裸钢板坯，演示变成底盘横梁或车身加强件 | https://commons.wikimedia.org/wiki/File:Brammenlager1.jpg |
